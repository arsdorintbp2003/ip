# Arsdorint chatbot

This is Arsdorint chatbot, a member of Arsdorint Team, and an adaptation application of CS2113 Project Duke) 

## User Guide

1. To download the application, click on this link.
2. The chatbot contains the following feature
 + List
 + Todo
 + Deadline
 + Event
 + Mark
 + Unmark
 + Date
 + Find
 + Bye
